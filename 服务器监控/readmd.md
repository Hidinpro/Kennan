库：
	监控者 monitor.h monitor.c 会被封装成libmonitor.so库
	被监控者 monitored.h monitored.c 会被封装成libmonitored.so库
	查看者 viewers.h viewers.c 会被封装成libviewers.so库
	
监控者：
	1、创建消息队列、共享内存，用于与监控者通信
	2、线程1
		从消息队列接收消息
		注册：
			判断总量 > 0
			遍历内存寻找空位置
			总量--
			标志使用
			返回空位置下标
		注销：
			取消使用标志
			总量--
	3、线程2
		通过网络接收数据
		查询：
			接收：程序名+变量名
			返回：下标
		访问：
			接收：下标
			返回：数据
	4、定时器
		每隔n秒写入所有数据的值到日志文件
		每隔n秒写入所有数据的值到数据库	
被监控者：
	初始化监控功能：获取消息队列、共享内存
	
	注册监控数据：
		通过消息队列发送数据给监控者(我是谁+变量名),监控者会返回一个下标。
		
	根据下标 数据加
	根据下标 数据减
	
	注销监控数据:
		把要注销的下标发送给监控者
		
查询者：
	发送程序名+变量名，获取变量的下标
	for(;;)
	{
		发送下标进行查询
		接收变量的值
		sleep(n)
	}
